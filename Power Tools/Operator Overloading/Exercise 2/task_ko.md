## 연산자 오버로딩 (#2)

다음 코드로 시작하세요:

```kotlin
enum class Count {
  Eeny, Meeny, Miney, Moe;
  companion object {
    val max = Moe
  }
}
```

`Count`에 대해 `++`와 `--` 연산을 수행하는 확장 함수를 작성하세요.  
`++` 연산이 `Moe`를 넘어가려고 하면 `Eeny`로 다시 돌아가고,  
`--` 연산이 `Eeny` 아래로 가려고 하면 `Moe`로 다시 돌아가도록 구현하세요.  
`main()` 내에 제공된 초기 코드가 여러분의 작업을 확인할 것입니다.

<sub> 이 과제는 자동 테스트를 포함하지 않으므로,  
"체크"를 실행할 때 항상 "정답"으로 표시됩니다.  
제공된 솔루션과 여러분의 솔루션을 비교해보세요! </sub>
